# SDBuildUtils
SDBuildUtils is a minecraft server plugin by [Snake Development](https://github.com/Snake-Development) that adds several survival-focused building utilities.

## Installation

Go to the `Releases` tab and download the `SDBuildUtils.jar` file from latest release.

## Usage

- Hold a blaze rod in game
- right click to set position 1
- right click again to set position 2
- do `/selection cancel` to cancel selection
- do `/selection resize <face> <operation> <amount>` to resize the selection
- do `/selection build <block> <delay_per_block>` to place blocks with custom delay
