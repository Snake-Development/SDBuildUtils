### Installation:

- Go to `Releases` tab
- Go to latest release and download the `SDBuildUtils.jar` file

### Usage:

- Hold a blaze rod in game
- right click to set position 1
- right click again to set position 2
- do `/selection build <block> <delay_per_block>` to place blocks with custom delay
- do `/selection cancel` to cancel the selection
